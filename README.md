# Student-Performance-predictor
ML project using streamlit

A Machine Learning project that predicts student scores based on study hours and attendance using a simple Streamlit web app.

---

## ⚙️ Tech Stack
Python • Pandas • Scikit-learn • Streamlit  

---

## 🚀 Run Project
streamlit run app.py

---

## 📈 Model
Random Forest Regressor
